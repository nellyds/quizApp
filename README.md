# quiz_app
A quiz app using questions loaded from an included json.  
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve to deploy the application locally.
```


### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
